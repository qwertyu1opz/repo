# некромантия ios 5 🔮

> Cydia-репозиторий для классических устройств Apple под управлением iOS 5 и выше.

## 📥 Как добавить репозиторий в Cydia

1. Откройте **Cydia** на вашем устройстве.
2. Перейдите во вкладку **Источники** (Sources) внизу экрана.
3. Нажмите кнопку **Правка** (Edit) в правом верхнем углу, а затем **Добавить** (Add) в левом.
4. Введите следующий адрес репозитория:
   ```text
   https://qwertyu1opz.github.io/repo/
   ```
5. Нажмите **Добавить источник** (Add Source) и дождитесь окончания обновления.
