# некромантия ios 5 🔮

> Cydia-репозиторий темной магии для классических устройств Apple под управлением iOS 5 и выше.

Этот репозиторий настроен для автоматической работы в качестве статического Cydia-источника (APT-репозитория).

## 📥 Как добавить репозиторий в Cydia

1. Скопируйте ссылку на ваш репозиторий (например, `https://username.github.io/repo/`).
2. Откройте **Cydia** на вашем устройстве.
3. Перейдите во вкладку **Источники** (Sources) -> **Правка** (Edit) -> **Добавить** (Add).
4. Введите адрес репозитория и нажмите **Добавить источник**.

## 🛠️ Как управлять репозиторием и добавлять пакеты

В репозитории настроена автоматизация. У вас есть два способа добавлять `.deb` файлы:

### Способ 1: Локально через скрипт

1. Поместите ваши файлы `.deb` в папку `debs/`.
2. Запустите скрипт генерации индексов:
   ```bash
   python3 update.py
   ```
3. Скрипт создаст/обновит файлы индексов `Packages`, `Packages.gz`, `Packages.bz2` и `Release`.
4. Сделайте коммит и отправьте изменения в GitHub:
   ```bash
   git add .
   git commit -m "Добавлены новые пакеты"
   git push
   ```

### Способ 2: Автоматически через GitHub Actions (рекомендуется)

1. Просто добавьте новые файлы `.deb` в папку `debs/` (вы можете сделать это даже прямо через веб-интерфейс GitHub).
2. Сделайте push/сохраните изменения.
3. GitHub Actions автоматически запустит `update.py` на сервере, пересоберет файлы индексов и закоммитит их обратно в репозиторий.
