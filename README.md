# некромантия ios 5 

> Cydia-репозиторий в стиле **iOS 5** для классических устройств Apple (iOS 5 и выше).

Сайт-витрина оформлен в аутентичном скевоморфном стиле iOS 5: льняной фон,
глянцевые синие навигационные панели, сгруппированные списки в духе «Настроек»
и глянцевая кнопка **«Добавить в Cydia»**.

## Добавить в Cydia

Откройте сайт репозитория на устройстве и нажмите кнопку **«Добавить в Cydia»**:

```text
https://qwertyu1opz.github.io/repo/
```

Или вручную:

1. Откройте **Cydia**.
2. Вкладка **Источники** (Sources) внизу.
3. **Правка** (Edit) → **Добавить** (Add).
4. Введите адрес выше и нажмите **Добавить источник**.

## Добавление пакетов

1. Положите `.deb` файлы в папку `debs/`.
2. Запустите `python3 update.py`.
3. Закоммитьте обновлённые `Packages*` и `Release`.
