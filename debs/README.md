# Папка для .deb пакетов / Deb Packages Folder

Поместите ваши готовые `.deb` твики/темы в эту папку.

## Как обновить репозиторий:

1. Скопируйте ваши `.deb` файлы сюда (в папку `debs/`).
2. Запустите скрипт `update.py` в корне репозитория:
   ```bash
   python3 update.py
   ```
3. Скрипт автоматически:
   - Распакует и прочитает файлы `control` из ваших дебов.
   - Подсчитает размеры и хэши (MD5, SHA1, SHA256) файлов.
   - Сгенерирует файлы `Packages`, `Packages.gz` и `Packages.bz2` в корне.
   - Обновит файл `Release` с правильными контрольными суммами.
4. Отправьте изменения в GitHub:
   ```bash
   git add .
   git commit -m "Добавлены новые заклинания"
   git push
   ```
5. Включите **GitHub Pages** в настройках вашего репозитория (укажите ветку `main`/`master` и корень `/root`), чтобы сайт репозитория был доступен в сети!
